# Vercel

> Vercel is an AI Cloud platform that provides frontend developers with the frameworks, workflows, and infrastructure to build, deploy, and scale web applications and AI-powered experiences.

Vercel was founded by Guillermo Rauch, creator of Next.js and Socket.io. The platform is known for its seamless Git integration, automatic deployments, serverless compute, and support for Jamstack architecture. Vercel also develops and maintains the Next.js React framework and acquired Turborepo in 2021.
